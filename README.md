Overview of QUsb2Snes and SNI
- Both QUsb2Snes and SNI are tools designed to enhance the experience of playing SNES games, particularly in the context of randomizers and multiplayer setups. They serve different purposes and have unique features.

QUsb2Snes
Key Features
- Websocket Server: Acts as a multi-platform framework for USB2SNES, allowing various applications to communicate with SNES hardware.
- Compatibility: Supports SD2SNES with USB2SNES firmware, emulators using Emulator Network Access protocol, and modded SNES Classics.
- Functionality: Enables uploading ROMs, reading and writing console memory, and managing files on the SD card.
- Limitations: Only one application can access the SD2SNES at a time, which can restrict simultaneous functionalities.

SNI
Key Features
- Network Control: Provides a way for emulators to connect and communicate, primarily for tracking items in randomizers.
- Compatibility: Works with specific emulators like Snes9x-nwa, BSNES-plus, and BizHawk.
- Integration: Automatically included with Archipelago installations, making it easy to set up for multiworld games.
- Limitations: Not compatible with QUsb2Snes, which can restrict users from using both tools together.
